# burgers-api

## Prerequisites
1. Installed Node.js
2. Installed MongoDB

## Setup
1. Clone project 
  ```sh
   git clone https://github.com/vikinatora/foodlance-delivery.git
   ```
2. Install dependencies 
  ```sh
   npm install
   ```
3. Seed database
 ```sh
   npm run seed
   ```
4. Run in dev mode
 ```sh
   npm run dev
   ```
